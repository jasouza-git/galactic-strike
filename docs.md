## `engine` class
- **config** - Configurations of engine
- **status** - Status of engine
  - `0` - Inactive
  - `1` - In transition
  - `2` - Error
  - `3` - Simulating
  - `4` - Client
- **listen** - Event listeners
- **debugs** - Invalid use of engine
